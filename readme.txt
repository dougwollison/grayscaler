Adds additional image processing to WordPress, creating grayscale versions of images during upload.

== Installation ==

1. Upload the contents of `grayscaler.tar.gz` to your `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
